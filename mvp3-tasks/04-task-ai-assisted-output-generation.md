# Task 04: AI 산출물 생성 지원 API

## 목표

요구사항 입력을 받아 업무 개념 추출, 데이터 사전 매핑, 신규 후보, 표준 위반 경고를 생성한다.

## OpenAPI 필수 조건

AI 산출물 생성 API는 구현 전에 OpenAPI Spec을 먼저 작성해야 한다.

## 상세 태스크

1. `POST /ai/development-assist` OpenAPI Spec을 작성한다.
2. 요구사항 텍스트, 대상 산출물, 도메인 옵션을 요청에 포함한다.
3. 업무 개념 추출 결과 schema를 정의한다.
4. 데이터 사전 매핑 결과 schema를 정의한다.
5. 생성 산출물 placeholder schema를 정의한다.
6. 신규 용어 후보 schema를 정의한다.
7. 표준 위반 경고 schema를 정의한다.
8. 요구사항에서 업무 개념 후보를 추출한다.
9. 정확 검색, 유사어 검색, 의미 검색을 순차 적용한다.
10. 표준 용어 매핑 결과를 생성한다.
11. 신규 용어 후보를 생성한다.
12. Rule Engine으로 표준 위반을 검증한다.
13. DTO, OpenAPI Schema, SQL 예시 생성을 지원한다.
14. contract test를 작성한다.

## 산출물

- AI Assist API OpenAPI Spec
- AI 산출물 생성 지원 API
- 업무 개념 추출 로직
- 표준 용어 매핑 결과

## 완료 기준

- `고객별 주문 내역을 조회하는 API 만들어줘.` 입력 시 고객번호, 주문번호, 주문일자, 주문금액, 주문상태코드 매핑을 반환할 수 있다.
- 반환 결과는 임의 용어를 확정하지 않고 신규 후보를 분리한다.
