# Task 03: AI 프롬프트 템플릿 관리

## 목표

바이브코딩과 기획서 작성에 사용할 데이터 사전 기반 프롬프트 템플릿을 관리한다.

## 범위

- 바이브코딩용 프롬프트
- 기획서 작성용 프롬프트
- 템플릿 버전 관리
- 템플릿 조회 API

## 상세 태스크

1. 바이브코딩용 프롬프트 템플릿을 등록한다.
2. 기획서 작성용 프롬프트 템플릿을 등록한다.
3. 템플릿 변수 목록을 정의한다.
4. 템플릿 버전 정책을 정의한다.
5. 템플릿 저장 구조를 구현한다.
6. 템플릿 조회 API OpenAPI Spec을 작성한다.
7. 템플릿 조회 API를 구현한다.
8. 템플릿 미리보기 기능을 구현한다.
9. 데이터 사전 검색 결과를 템플릿 변수로 주입하는 방식을 구현한다.
10. 템플릿 변경 이력을 기록한다.

## 산출물

- 프롬프트 템플릿
- 템플릿 저장 구조
- 템플릿 조회 API OpenAPI Spec
- 템플릿 조회 API

## 완료 기준

- AI 개발 도구가 표준 용어 규칙 프롬프트를 조회할 수 있다.
- 템플릿 버전을 추적할 수 있다.
