# Task 04: 용어 관리자 MVP 화면 구현

## 목표

관리자가 표준 용어를 검색, 조회, 등록, 수정할 수 있는 MVP 화면을 구현한다.

## 범위

- 용어 검색 화면
- 용어 상세 화면
- 용어 등록/수정 화면
- 표현 매핑 영역
- 별칭/금지어 영역

## 상세 태스크

1. 용어 검색 화면 레이아웃을 구현한다.
2. 한글명, 영문명, 약어, API 필드명 검색 입력을 제공한다.
3. 검색 결과 테이블을 구현한다.
4. 용어 상세 화면을 구현한다.
5. 업무 정의, 사용 맥락, 물리 타입, 자릿수, 소수점을 표시한다.
6. DB 컬럼명, API 필드명, 코드 변수명, UI 라벨을 표시한다.
7. 별칭, 유사어, 금지어 목록을 표시한다.
8. 용어 등록 화면을 구현한다.
9. 용어 수정 화면을 구현한다.
10. API 호출 타입은 OpenAPI Spec에서 생성된 클라이언트 타입을 우선 사용한다.
11. 상태값은 Draft, Reviewing, Approved, Deprecated, Rejected를 일관되게 표시한다.

## 산출물

- 용어 검색 화면
- 용어 상세 화면
- 용어 등록/수정 화면
- OpenAPI 기반 API 클라이언트 연동

## 완료 기준

- 관리자가 UI에서 고객번호 용어를 등록하고 상세 조회할 수 있다.
- UI에서 고객ID 별칭을 확인할 수 있다.
