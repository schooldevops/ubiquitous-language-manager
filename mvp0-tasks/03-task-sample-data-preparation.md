# Task 03: 기존 데이터 사전 샘플 확보 및 정리

## 목표

MVP 검증에 사용할 기존 데이터 사전 샘플을 확보하고 정비한다.

## 범위

- 기존 데이터 사전 샘플 수집
- 필수 컬럼 매핑
- 중복 용어 확인
- 중복 약어 확인
- 폐기 후보 용어 식별

## 상세 태스크

1. 기존 데이터 사전 샘플 파일을 확보한다.
2. 기존 컬럼을 `TERM_MASTER` 후보 컬럼과 매핑한다.
3. 한글 용어명 중복을 식별한다.
4. 영문 약어 중복을 식별한다.
5. 물리 타입 누락 항목을 식별한다.
6. 자릿수, 소수점 누락 항목을 식별한다.
7. 관련 시스템 누락 항목을 식별한다.
8. 폐기 후보 용어를 분리한다.
9. MVP 샘플 용어 세트를 만든다.
10. 고객번호, 주문번호, 주문일자, 주문금액, 주문상태코드 예시를 포함한다.

## 산출물

- 데이터 사전 샘플
- 컬럼 매핑표
- 데이터 품질 점검 결과
- MVP 샘플 용어 목록

## 완료 기준

- MVP 개발과 테스트에 사용할 샘플 용어가 준비되어 있다.
- 고객ID 입력 시 고객번호를 추천하는 테스트 데이터를 만들 수 있다.
