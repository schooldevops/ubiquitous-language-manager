# Task 01: Graphify 그래프 모델 설계

## 목표

용어 관계와 영향도 분석을 위한 그래프 노드 및 관계 모델을 설계한다.

## 범위

- 그래프 노드 정의
- 그래프 관계 정의
- 관계 방향 정의
- 관계 속성 정의
- 동기화 기준 정의

## 상세 태스크

1. `Term` 노드를 정의한다.
2. `Domain` 노드를 정의한다.
3. `System` 노드를 정의한다.
4. `Table` 노드를 정의한다.
5. `Column` 노드를 정의한다.
6. `API` 노드를 정의한다.
7. `APIField` 노드를 정의한다.
8. `DTO`와 `Entity` 노드를 정의한다.
9. `Document`와 `TestCase` 노드를 정의한다.
10. `represents`, `mentions`, `uses`, `belongsTo`, `deprecatedBy` 관계를 정의한다.
11. 그래프 동기화 키를 정의한다.
12. 삭제 또는 폐기 데이터 처리 정책을 정의한다.

## 산출물

- Graphify 그래프 모델 문서
- 노드 정의표
- 관계 정의표
- 동기화 정책

## 완료 기준

- 고객번호와 CUST_NO, customerNumber, 관련 API 필드를 그래프로 연결할 수 있는 모델이 정의되어 있다.
